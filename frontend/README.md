# Exam Problem Extractor Frontend

React frontend application for the Exam Problem Extractor API.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```bash
VITE_API_BASE_URL=http://localhost:8000
```

3. Run development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`.

## Build

```bash
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- React Router
- Tailwind CSS
- Axios
